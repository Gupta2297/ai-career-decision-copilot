# AI Career & Decision Copilot

A multi-mode AI copilot designed to help with high-impact decisions including interview preparation, networking conversations, task prioritization, and industry intelligence.

## 🧠 Why I Built This

Most AI tools generate answers.

This project was built to do something different:

👉 Turn unstructured inputs into **clear, actionable decisions**

Instead of just responding, the system:
- structures thinking
- prioritizes actions
- provides ready-to-use outputs

---

## ⚙️ Core Features

### 🟦 Interview Mode
- Generates structured, personalized answers  
- Uses real experience (Velera, Pick N Drop)  
- Focuses on decision-making, not just storytelling  

---

### 🟩 Time & Priority Mode
- Breaks down tasks  
- Assigns priority  
- Recommends execution order  

---

### 🟨 Networking Mode
- Prepares conversation flow  
- Generates high-signal questions  
- Helps position user effectively  

---

### 🟥 Industry Intelligence Mode
- Summarizes trends  
- Explains impact  
- Recommends actions  

---

## 🧩 How It Works

The system is built using:

- **Claude Projects** for structured AI behavior  
- Custom **mode-based instructions**  
- Context-aware responses grounded in real experience  
- A front-end **Artifact UI** for usability  

Each response follows a structured format:
1. Key Insight  
2. Final Answer (usable immediately)  
3. Structured Breakdown  
4. Recommended Next Action

---

## 🛠️ How I Built This

- Designed a mode-based AI system using Claude Projects  
- Implemented structured system instructions for consistent outputs  
- Integrated user context (CV-based reasoning) for personalization  
- Built a front-end Artifact UI for usability and demo purposes  
- Iterated on output structure to prioritize actionable responses over generic answers  

---

## 💡 Key Product Decisions

- Mode-based system instead of generic chat  
- Prioritized **usable output first**, not long explanations  
- Designed for **real-world scenarios**, not demo prompts  
- Built to work with or without user-provided context
